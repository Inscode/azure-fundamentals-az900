# Consumption-Based Model 

## Definition
The consumption-based model is a pricing model used in cloud computing where you **pay only for the resources you use**, similar to utilities like electricity or water.

There is **no upfront cost**, and billing is based on actual usage.

---

## Key Characteristics
- Pay-as-you-go pricing
- No upfront investment
- Costs depend on usage
- Scale resources up or down at any time
- Stop paying when resources are stopped or deleted

---

## How It Works in Cloud Computing
- You create cloud resources (VMs, storage, databases)
- You are billed based on:
  - Time used (hours/minutes)
  - Amount of storage used
  - Data transferred
- When usage stops, billing stops (for most services)
