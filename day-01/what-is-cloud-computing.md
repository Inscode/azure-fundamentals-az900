# What is Cloud Computing?

## Definition
Cloud computing is the delivery of computing services such as **servers, storage, databases, networking, software, and analytics** over the internet (the *cloud*) instead of using local physical hardware.

These services are provided on a **pay-as-you-go** basis, meaning you pay only for what you use.

---

## Traditional IT vs Cloud Computing

### Traditional IT (On-Premises)
- Buy physical servers and hardware
- High upfront cost
- Limited scalability
- Maintenance handled by the organization

### Cloud Computing
- No need to buy physical hardware
- Pay only for usage
- Easy to scale up or down
- Maintenance handled by the cloud provider
