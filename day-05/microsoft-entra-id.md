## What is Microsoft Entra ID?
Microsoft Entra ID (formerly Azure Active Directory) is a cloud-based identity and access management service that helps organizations manage users, applications, and access to resources securely. It provides authentication, authorization, and policy enforcement across cloud and on-premises environments.

Think of it as:
> The identity system that verifies who you are and controls what you can access in Azure and connected applications.

---

## Why Microsoft Entra ID Matters
- Provides secure user identity management
- Enables centralized access control
- Supports cloud and hybrid environments
- Protects applications and data
- Simplifies user authentication

---

## Key Features

- **Single Sign-On (SSO)** — Access multiple applications with one login.
- **Multi-Factor Authentication (MFA)** — Adds extra security verification.
- **Conditional Access** — Enforces policies based on user, location, or device.
- **B2B Collaboration** — Allows external partners to securely access resources.
- **Application Management** — Integrates and manages application access.
- **Device Management** — Controls and secures device access.
