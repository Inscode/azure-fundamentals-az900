# Microsoft Defender for Cloud — Notes (Azure Focus)

## Overview
- Microsoft Defender for Cloud is a cloud security posture management (CSPM) and cloud workload protection platform (CWPP).
- Provides unified security across:
  - Azure
  - On-premises environments
  - AWS
  - Google Cloud
- Helps detect threats, strengthen security posture, and protect workloads.

## Core Capabilities (Detect • Deter • Delay)
- **Detect** — Continuous monitoring and threat detection.
- **Deter** — Security recommendations and hardening guidance.
- **Delay** — Controls that slow or prevent attacker movement.

## Integrations
- GitHub — Secure code and repositories.
- Azure — Native protection for Azure resources.
- DevOps — Pipeline security.
- GitLab — Security insights for CI/CD.

## Key Benefits
- Comprehensive security visibility across environments.
- Centralized security management.
- Automated recommendations to improve posture.

## Use Cases
- Cloud workload protection (VMs, containers, databases).
- Compliance monitoring and regulatory reporting.
- Strengthening overall cloud security posture.

## Why It Matters in Azure
- Continuously assesses Azure resources against best practices.
- Provides Secure Score to measure security improvements.
- Helps detect misconfigurations and active threats early.

## Quick Summary
- Single pane of glass for multi-cloud security.
- Combines posture management + threat protection.
- Essential service for securing Azure environments.
